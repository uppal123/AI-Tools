# AI-Tools
Collection of AI powered QA tools: 
1. Bug Snap – Create Jira tickets from screenshots
    Drag and drop screenshot and ask tool to generate jira ticket out of it

2. Exploratory Charter Generator
   Allow QA engineers to enter a product URL and automatically generate structured
   exploratory testing charters using AI.
